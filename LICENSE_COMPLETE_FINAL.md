## 💡 版权保护机制详解

### 技术实现

**组件**: `LicenseFooter.tsx`

**工作原理**:
1. 组件挂载时调用 `/api/license/info`
2. 检查 `licensed` 状态
3. 如果 `licensed: false` → 显示版权条
4. 如果 `licensed: true` → 返回 null（不显示）
5. 每5分钟自动重新检查

**关键代码**:
```typescript
// 如果已授权，不显示任何版权信息
if (licenseInfo?.licensed) {
  return null;
}

// 未授权时显示版权信息
return <div>版权提示条...</div>
```

### 自动更新机制

```typescript
useEffect(() => {
  fetchLicenseInfo();
  // 每5分钟检查一次
  const interval = setInterval(fetchLicenseInfo, 5 * 60 * 1000);
  return () => clearInterval(interval);
}, []);
```
