## 💡 使用示例

### 初始化授权系统
```typescript
import { LicenseManager } from '@/lib/license';

// 应用启动时
await LicenseManager.initialize();
```

### 激活授权
```typescript
const license = await LicenseManager.activate('LIC-2024-XXXXX');
console.log(`授权已激活：${license.plan}`);
```

### 验证授权
```typescript
const result = await LicenseManager.verify();
if (result.valid) {
  console.log('授权有效');
} else {
  console.error(result.errorMessage);
}
```

### 检查功能
```typescript
if (LicenseManager.hasFeature('ai')) {
  enableAI();
}
```
