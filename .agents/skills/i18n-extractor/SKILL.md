name: i18n-extractor
description: 扫描指定的页面或组件，提取代码中的硬编码中文字符串，自动转化为基于 i18n 多语言机制的引用，并生成对应的语言包 JSON 文件字典。
---

# 任务目标
辅助项目的国际化改造。自动识别组件或页面中硬编码（Hard-coded）的中文文本，将其替换为获取国际化词条的调用形式，并将提取出的文本按键值对统一管理。

# 执行步骤
1. 根据用户指定的目录或单个文件（如 `src/components/ui/`），提取文件内容。
2. 使用正则匹配或语法树分析找出在 JSX/TSX 中用于显示的中文硬编码文案。
3. 把这些文案抽象出具有语义化的 `key`，例如：`"确认保存"` -> `button.confirmSave`。
4. （直接修改代码）在文件中引入您的多语言 Hook（如 `const { t } = useTranslation();`），把原本的 `"确认保存"` 替换为 `{t('button.confirmSave')}`。
5. （更新字典）将新识别的键值对合并到诸如 `src/locales/zh-CN.json` 和 `src/locales/en-US.json` 这样的多语言配置文件中。对于外语则使用基于 AI 的准确机器翻译作初始填充。
6. 返回处理完毕的文件列表和新增/更新的国际化词条总数供审核。
