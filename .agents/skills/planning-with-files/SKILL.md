# 规划与文件跟踪技能 (Planning with Files)

这个技能赋予了 AI 开发助手使用工程目录内的文件文件进行长期任务跟踪的“工作记忆”超能力，让你能够在复杂的任务和长时间的开发上下文中，维持高度的上下文一致性和任务控制力。

## 使用规则

1. **创建计划档**: 当用户布置了一个复杂的、多步骤的系统级任务时，不要急于开始编写代码。首先使用 `write_to_file` 工具创建一个 `.agents/PLAN.md` 计划文件。
2. **渐进式任务拆解**:
   - 在 `PLAN.md` 中，使用标准的 Markdown Todo 列表风格（如 `- [ ] 步骤1`）。
   - 将主任务深度拆分为各个模块和函数级别的可执行项。
3. **实时更新状态**: 每当完成某个子步骤时，**必须**回头修改 `.agents/PLAN.md`，将对应项标记为完成（`- [x] 步骤1`），并可以在下方简要记录踩过的坑或得到的输出。
4. **断点续传**: 当对话轮次重置，或者你在继续之前的任务时，应该优先使用 `view_file` 读取 `PLAN.md` 文件，以便瞬间恢复你的工作上下文和接下来的目标。
5. **复盘收尾**: 任务结束后，在 `PLAN.md` 中写一段简短的复盘，或者提示用户该阶段计划已闭环。
