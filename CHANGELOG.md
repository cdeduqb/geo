# 更新日志 (Changelog)

## v0.3.14 (2026-01-11)

### 核心修复
- **内容发布**：修复了在生产环境（服务器）上，创建、更新或删除页面内容后，前台页面无法立即刷新的问题（增强了 ISR 缓存刷新机制）。

## v0.3.13 (2026-01-11)

### 体验优化
- **页面管理**：优化了 URL 冲突时的错误处理，现在会通过 Toast 提示告知用户，而不是通过全屏报错打断流程。

## v0.3.12 (2026-01-11)

### 修复
- **页面管理**：修复了创建重复 URL 页面导致系统崩溃的问题，增加了 URL 唯一性校验和友好的错误提示。

## v0.3.11 (2026-01-11)

### 修复
- **验证缓存清除**：禁用了站点验证文件的浏览器缓存，解决添加或修改验证文件后无法立即生效的问题。

## v0.3.10 (2026-01-11)

### 体验优化
- **即时更新**：修复了修改站点设置后首页未立即刷新的问题 (Router Cache 配置优化)。

## v0.3.9 (2026-01-11)

### 修复
- **验证参数传递**：修复了在中间件转发时导致的文件名参数丢失问题 (`File Not Found`)，增加了 Header 传参作为双重保障。

## v0.3.8 (2026-01-11)

### 重构
- **核心优化**：将站点验证机制升级为中间件动态拦截 (Middleware) 模式，不再依赖服务器文件写入权限，极大提升了在各类部署环境下的稳定性和成功率。

## v0.3.7 (2026-01-11)

### 优化
- **调试增强**：增加了站点设置保存时的文件写入日志，便于排查验证文件生成失败的问题。

## v0.3.6 (2026-01-11)

### 新增功能
- **站点所有权验证**：新增站点验证文件管理功能，支持直接导入 HTML 验证文件并物理同步到服务器，彻底解决搜索引擎验证失败的问题。
- **IndexNow 优化**：新增 IndexNow 密钥一键获取链接，指向官方生成器页面。

### 修复与优化
- **SEO 推送修复**：修复了在 IndexNow 和其他平台推送时使用了错误 URL (如 localhost) 的问题。
- **Google 推送提示**：优化了 Google Indexing API 的配置指引，明确了 Access Token 的有效期限制。
