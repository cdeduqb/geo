## 🚀 快速开始

### 步骤1: 访问客户管理

**从仪表板**:
```
http://localhost:3000/license-admin/dashboard
→ 点击"客户管理"
```

**直接访问**:
```
http://localhost:3000/license-admin/customers
```

### 步骤2: 创建客户

点击右上角的"创建客户"按钮

### 步骤3: 填写信息

**必填字段**:
- ✅ 邮箱 (customer@example.com)

**可选字段**:
- 公司名称 (如: 测试公司)
- 联系人 (如: 张三)
- 联系电话 (如: 13800138000)
- 地址 (如: 北京市朝阳区...)
- 状态 (激活/暂停/关闭)

### 步骤4: 提交创建

点击"创建客户"按钮

### 步骤5: 创建授权

创建成功后，访问授权生成页面：
```
http://localhost:3000/license-admin/generate
```

选择刚创建的客户，配置授权信息，生成授权码
