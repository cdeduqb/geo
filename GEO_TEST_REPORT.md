## 🐛 问题详情

### 问题1: AI API调用失败

**严重程度**: 中等  
**影响范围**: 质量评分功能完全不可用  
**用户影响**: 无法使用AI评分，但不影响其他GEO功能

**技术细节**:
```typescript
// 失败位置: src/app/api/admin/geo/score/route.ts:119
if (!response.ok) {
    throw new Error(`AI API调用失败: ${response.statusText}`);
}
```

**可能的原因**:
1. ✅ **AI配置问题** (最可能)
   - 数据库中的AIConfig配置不正确
   - API Key无效或已过期
   - baseUrl设置错误

2. ⚠️ **请求格式问题**
   - AI Provider不支持当前请求格式
   - 缺少必需的请求参数
   - Content-Type设置不正确

3. ⚠️ **网络问题**
   - 无法访问AI Provider的API
   - 防火墙阻止请求
   - 超时设置太短
