## 🔗 授权管理功能

访问路径有两种方式：

### 方式1: 侧边栏入口（新增）
```
系统设置 → 授权管理
```

### 方式2: 直接访问
```
http://localhost:3000/admin/license
```

### 功能说明
- 查看当前授权状态
- 激活新授权
- 显示授权详情（套餐、有效期、域名等）
- 未授权时显示提示
